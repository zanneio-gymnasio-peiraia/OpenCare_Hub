# OpenCare_Hub
Είμαστε η STEAM Lab team από το Ζάννειο Πρότυπο Γυμνάσιο Πειραιά

Πειραματιζόμαστε με προγραμματισμό, αλγορίθμους και STEAM projects, με στόχο τη μάθηση και τη δημιουργία.  

Αποθετήριο για την συμμετοχή στον 8ο Πανελλήνιο Διαγωνισμό Ανοιχτών Τεχνολογιών στην Εκπαίδευση (https://openedtech.ellak.gr/)
